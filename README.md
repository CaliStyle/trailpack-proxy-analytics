Analytics for Proxy Engine.

## Install

```sh
$ npm install --save trailpack-proxy-analytics
```

## Configure

```js
// config/main.js
module.exports = {
  packs: [
    // ... other trailpacks
    require('trailpack-proxy-analytics')
  ]
}
```
